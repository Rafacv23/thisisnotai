# Contributing to thisisnotai

Thank you for your interest in contributing to **thisisnotai**!
We welcome all kinds of contributions—code, translations, documentation, ideas, and more.

---

## How to Contribute

1. **Fork the repository**
   Click the "Fork" button at the top right of this page.

2. **Clone your fork**
   ```sh
   git clone https://github.com/rafacv23/thisisnotai.git
   cd thisisnotai
   ```

3. **Create a new branch**
   ```sh
   git checkout -b my-feature
   ```

4. **Make your changes**
   - For code: Follow the project’s style and conventions.
   - For translations: Edit the relevant JSON files in `/messages`.
   - For documentation: Update or add Markdown files.

5. **Test your changes**
   - Run `npm run dev` or `bun run dev` to start the development server.
   - Make sure everything works as expected.

6. **Commit and push**
   ```sh
   git add .
   git commit -m "Describe your changes"
   git push origin my-feature
   ```

7. **Open a Pull Request**
   - Go to your fork on GitHub and click "Compare & pull request".
   - Fill in the PR template and submit.

---

## Guidelines

- **Be respectful**: This is a community for celebrating human creativity.
- **No AI-generated content**: All contributions must be original and human-made.
- **Write clear commit messages**.
- **Document your code** where helpful.
- **Ask questions**: If you’re unsure, open an issue or discussion.

---

## Need Help?

Open an [issue](https://github.com/rafacv23/thisisnotai/issues) or start a [discussion](https://github.com/rafacv23/thisisnotai/discussions).

Thank you for helping us keep this space truly human!
